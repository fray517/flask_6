# Flask приложение - Анкета пользователя

Простое веб-приложение на Flask для ввода и отображения данных о пользователе.

## Функциональность

- Форма для ввода данных о пользователе (имя, город, хобби, возраст)
- Отображение введенных данных на той же странице после отправки
- Современный интерфейс с использованием Bootstrap 5
- Шаблоны на основе Jinja2

## Установка

1. Активируйте виртуальное окружение:
```powershell
.\venv\Scripts\Activate.ps1
```

2. Установите зависимости:
```powershell
pip install -r requirements.txt
```

## Запуск

Запустите приложение:
```powershell
python app.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000
